# Severn LS Edge AI

A web-based platform for 5th grade students to connect Arduino BLE Sense devices, collect sensor data, and train machine learning models using Edge Impulse.

## Features

- Web Bluetooth API connection to Arduino BLE Sense
- Real-time accelerometer and proximity sensor data collection
- Student-friendly interface for ML model training
- Integration with Edge Impulse for model deployment

## Getting Started

Open the web page and click "Connect to Arduino" to begin collecting sensor data from your Arduino BLE Sense device.
